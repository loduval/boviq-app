Intégration BOVIQ V6

localStorage partagé · clé BOVIQ_MILKLIC

🔗 Principe

Ce module écrit dans localStorage['BOVIQ_MILKLIC']. BOVIQ V6 lit ce même localStorage pour enrichir les fiches animaux. Les deux fichiers doivent être ouverts dans le même navigateur / domaine.

1
Nav button dans BOVIQ V6 (après Taureaux) 
<button class="nav-btn" data-p="milklic">
  <span class="icon">🥛</span>Contrôle Laitier
</button>
2
Dans openAnimalModal() — jointure par boucle 
const mlRaw = localStorage.getItem('BOVIQ_MILKLIC');
if (mlRaw) {
  const ml = JSON.parse(mlRaw);
  const bNorm = (a.boucle||'').replace(/\s/g,'');
  const mlA = Object.values(ml.animaux).find(x =>
    (x.numNat||'').replace(/\s/g,'') === bNorm
  );
  if (mlA) renderMilklicBadge(mlA);
}
